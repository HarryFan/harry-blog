## Design Context

### Users
- 潛在合作對象與用人單位：快速理解作者能力、履歷入口、作品輸出品質
- 技術讀者：閱讀文章、用 RSS 追蹤、以文章判斷專業深度與風格
- Podcast 聽眾：找到《妖你聽新聞》入口並建立作者連結

### Brand Personality
- 三個關鍵字：清晰、克制、有稜角
- 情緒目標：專業可信、讀起來順、偶爾帶點幽默但不喧賓奪主

### Aesthetic Direction
- 方向：編輯感極簡（偏專欄／雜誌排版）
- 特色：強層級、留白、可讀性優先；用少量強色作為導覽與互動提示
- 內容視覺：文章封面缺圖時使用一致的 SVG 漸層 fallback

### Design Principles
- 讓內容先贏：標題、摘要、日期、閱讀節奏優先於裝飾
- 一眼可導航：導覽精簡、主路徑明確（Blog / About / Resume / Podcast）
- 用少量高品質連結：避免塞滿社群 icon，以清楚文字連結取代
- 快、穩、可維護：不引入大套 UI 框架，優先使用現有 CSS 與元件
